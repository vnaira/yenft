YENF Editor

Our innovative graphic editor can help you to design and release your NFT collection without any programming knowledge. Rarity calculation, prices calculation, mint details, ability to setup reveal date and much, much more - the system will take care of everything.

Components System

A component has a “Main Component” that serves as the parent and “instances” or children that are placed on the artboard. When changing the Main Component, the changes propagate to all of its instances. Based on the quantity of main and sub components will be generated the set of arts, that can be minted into NFT later.

NFT Generation Tool

You select the components, set rarity percentage on the traits you want to have, and Yenft generates your collection. The system supports up to 100,000 art pieces within a single NFT collection.
